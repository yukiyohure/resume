## 自己紹介

工業高校の情報系学科を卒業後、自動車製造業に入社し10ヶ月間ライン製造業の経験を積んだのち退職。その後、フィリピンにて3ヶ月のIT×語学留学を経験しています。  
卒業後は同校で英語を使った業務のインターンを半年間経験後、東京にてエンジニアとしてキャリアをスタートしました。  
エンジニアとして働きつつ、業務やプライベートで英日通訳・翻訳をしたり日本人エンジニアに向けて英語の勉強会を開くなど、英語に関する活動も続けてきました。

### ソフトスキル

- 「いい開発チーム・文化づくり」に興味があり、勉強会を実施したりランチ会などを通して間接的にチームメンバーのパフォーマンス向上に貢献できる
- 英語を用いたコミュニケーションや英語の一次資料を業務で活用することに抵抗がない
- 相手の意見と相手の人間性を切り分けて、建設的な議論ができる
- 手段と目的を混同せず、目的に沿った効果的な戦略を立てることができる

- デザインデータを用いてステークホルダー (プロダクトオーナー) への仕様を基にしたプロトタイプUIの提案
